# React Starter Kit
[![Build Status](https://travis-ci.org/xiaofan2406/react-starter-kit.svg?branch=master)](https://travis-ci.org/xiaofan2406/react-starter-kit)


### Start n Run
```
npm i

npm run dev # run dev server

npm run build # build for production
```
